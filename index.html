FUERZA BRUTA

Durante mucho tiempo, se necesitaron de n2 pasos de multiplicaciones de un solo dígito para multiplicar dos números de n dígitos. Para representar la forma de resolver una multiplicación por Fuerza Bruta, nos basamos en el método tradicional de resolver una multiplicación, en el apartado de Fuerza Bruta podemos observar el procedimiento que se sigue para llegar a la solución por medio de este camino.

n * n = n2 , donde n es el número de dígitos que tiene la cifra.
ALGORITMO DE KARATSUBA

El algoritmo de Karatsuba consiste en fragmentar la serie de dígitos de un número y recombinarlos de una manera nueva, de modo que un pequeño número de sumas y restas haga lo mismo que un gran número de multiplicaciones. La manera en la que se fragmentan los número es la siguiente:
x = a * 10n/2 + b           y = c * 10n/2 + d
x * y = (a * 10n/2 + b)(c * 10n/2 + d) = ac(10n/2) + (ad + bc) * 10n/2 + bd

Donde el término (ad + bc) = (a + b)(c + d) - ac - bd
